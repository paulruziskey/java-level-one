Project Two: Number Fortune Teller

Project Three ❯

In this project, you’ll use conditional expressions to tell some fortunes! Write code that asks the user for a number between one and 10. You must tell at least four fortunes, but you can tell up to 10 if you like! The code for this project currently tells five with users getting one fortune for all numbers less than four, another fortune for all numbers greater than six, and unique fortunes for numbers four, five, and six. You can use some of the example fortunes, or pick your own! Below you’ll find examples of the program handling bad input as well as the program telling fortunes.
